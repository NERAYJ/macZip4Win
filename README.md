# maczip2win

Finder の右クリックから、Windows で文字化けしにくい ZIP を作るための Automator クイックアクションです。

## 使い方

1. Automator を開く
2. 新規作成で「クイックアクション」を選ぶ
3. 画面上部の設定を以下にする

- ワークフローが受け取る現在の項目: `ファイルまたはフォルダ`
- 検索対象: `Finder`

4. 「シェルスクリプトを実行」アクションを追加
5. シェル: `/bin/zsh`
6. 入力の引き渡し方法: `引数として`
7. `windows-zip-automator.sh` の中身を貼り付け

## 動作

- 最初に選択した項目と同じ名前で ZIP を作成
- 同一フォルダ内の項目のみ対応
- Windows のエクスプローラで解凍しても文字化けしにくい形式

## 除外対象

- `.DS_Store`
- `._*`
- `__MACOSX`
- `.Spotlight-V100`
- `.Trashes`

## 注意

- 複数フォルダに跨る選択はエラーになります
- 既に同名の ZIP がある場合は上書きされます
